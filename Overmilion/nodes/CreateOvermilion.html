OvermilionExample > Overmilion > Create Overmilion

Create Overmilion

Creates an Overmilion object

Inputs

In
Exec

Configuration Name
String

Name of the configuration file to read/write (Example : Config.cfg)
Configuration Directory
String

Directory path where the configuration file is located/will be located
Load Settings
Boolean

If we should load the settings on object constructor
New Settings Rules
Overmilion Enabled Settings Structure

The settings to use and not to use, leaving this to null will result in Overmilion using defaults
Enable Autoexec
Boolean

Enable the autoexec module
Autoexec Settings
Overmilion Autoexec Settings Structure

Settings for the OvermilionAutoexec module, if enabled

Outputs

Out
Exec

Return Value
Overmilion Object Reference

A new Overmilion instance with the specified settings
